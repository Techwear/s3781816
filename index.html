Moda Inventario

Overview

My project will be a phone application of that contains the inventory of your personal closet. The application will act as a personal database that would be built with many features. The app will be called ‘Moda Inventario’ which is Italian for ‘fashion inventory’. The app will let the users store profiles for their item and sort them out into categories such as; tops, pants, shoes and bags. Upon storing the item into their profile, users can add descriptions, images, photos and favourite them. Moda Inventario will also have a clean and minimalistic design to suit the environment of boldness. The app is targeted towards people that are into fashion that want to label their clothes to give them an idea of what they have or for just anyone that wants to organise their belongings.

Motivation

The world’s clothing industry totals up to $2 trillion dollars and the shoe market is expected to reach $1 trillion dollars in 2019. This representation shows us the scale of clothing in everyday life and how we’re apart of it without even realising it. My project will be useful because it’ll give the people a lovely interface to organise their favourite pieces of clothes, shoes and accessories. The ability to have an inventory that you can access when you want is convenient for knowing what you own and for the ability of sharing your list with others if you please.

Description

Moda Inventario will be an application that will be available on apple and android devices. The applications goal is to allow users to create a database that will aesthetically list of their clothes, shoes and accessories. Upon opening the app, the user will be asked to log in or sign up with an email. This will allow users to sync their data through multiple devices. Once the user logs in they will be able to see their inventory which would be empty if they’re a new user. They will be given the option to add an item which they can manually input the data for or search up depending on the popularity of the item. For one item the user can add a name, type, brand, price, description of size or the item, colour pallet and photos to the item. Upon adding an item, they can sort them out into multiple categories such as; season, colour, type of item (shoes, bags, pants), descending or ascending price order, favourites and custom folders. From each category, the user will be able to copy the details they want from item categories for their own purpose, an example is for selling items to a Facebook page or just for sharing with friends. Another function would be able to link your social medias to this application which would beneficial for people that are trying to view, sell, buy or trade items. The app would allow people to view folders that you allow them to see, so for example if a user wanted to show their shoe collection on Instagram, they can link the folder to the page which will allow users to view the inventory on the app. On the app you have the option to allow people to follow you and follow other people for inspiration. Being able to follow different kinds of people around the world can give inspiration for you and others to influence their own style. But if users would rather use Instagram for that kind of function, they can decide to use the app for just their organisational needs. They will even be able to use an offline mode which allows them to view and add items to their inventory but syncing to your online profile requires internet connection to save the data to the server. Moda Inventario targets users from fashion enthusiasts to people that just want to organise their belongings.

Tools and Technologies

To program the application a reliable computer that can run software such as XCode or Microsoft Visual Studio would be ideal. Both allow me to construct the user interface and write code in them. Having knowledge in code language such as C++, Swift, Java or Python will allow the me to create the application. Apart from having the knowledge of code equipment such as a keyboard, monitor and mouse is all I need to create the application.

Skills Required

To create this project I must have the knowledge to code the application on developing software such as XCode or Microsoft Visual Studio. It easy to download developing software because they are normally free to the public. Learning to code the program will take time and practice but online tutorials are available online on YouTube and online forums.

Outcome

The outcome of my project’s success means that people will have another application to share their interests on. The application fixes the problem of closet organisation and labelling. Moda Inventario will impact communities that are interested in fashion.
